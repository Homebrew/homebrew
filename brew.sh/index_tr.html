OS X için eksik paket yöneticisi

  • Homebrew Apple'ın ihtiyaç duymadığı ama size lazım olabilecek şeyleri kurar.

    $ brew install wget

  • Homebrew paketleri kendi dizinlerine kurar ve dosyalarını kısayolla /usr/local'a bağlar.

    $ cd /usr/local
$ find Cellar
Cellar/wget/1.12
Cellar/wget/1.12/bin/wget
Cellar/wget/1.12/share/man/man1/wget.1

$ ls -l bin
bin/wget -> ../Cellar/wget/1.12/bin/wget

  • Homebrew kendisine belirtilen alan dışında yerlere kurulum yapmaz, ve Homebrew'i istediğiniz herhangi biryere koyabilirsiniz.

  • Basitçe kendi Homebrew paketlerinizi oluşturun.

    $ brew create http://foo.com/bar-1.0.tgz
Created /usr/local/Library/Formula/bar.rb

  • İçersinde tamamen git ve ruby'den oluşur, böylece değişikliklerinizi geri alabileceğiniz ve kaynağa doğru güncellemeleri birleştirebileceğiniz bilgisiyle kod yazabilirsiniz.

    $ brew edit wget # opens in $EDITOR!

  • Homebrew formülleri basit Ruby betikleridir:

    require 'formula'

class Wget < Formula
  homepage 'http://www.gnu.org/wget/'
  url 'http://ftp.gnu.org/wget-1.12.tar.gz'
  md5 '308a5476fc096a8a525d07279a6f6aa3'

  def install
    system "./configure --prefix=#{prefix}"
    system 'make install'
  end
end

  • Homebrew OS X'i tamamlar. Gem'lerinizi gem ile, ve bağlılıklarını brew ile yükleyin.

  • Homebrew'i yükleyin

    ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

    Bunu bir Terminal penceresine yapıştırın.

    Betik yapacaklarını açıklar ve yapmadan önce bekler. Burada (10.5 için gereklidir) başka kurulum seçenekleri bulunur.
